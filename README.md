Pandora API Ruby Client [WIP]
=============================

A Ruby wrapper for the [Pandora Tuner JSON API][tuner_api].

## Installation

Add this line to your application's Gemfile:

    gem 'pandora_client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pandora_client
